## Задание 1: Абитуриенты

Есть таблица `examination` со следующими полями:
- `id` — идентификатор абитуриента
- `scores` — количество баллов (0–100)

Необходимо присвоить каждому абитуриенту **уникальную позицию в рейтинге**, даже если баллы совпадают.

### Решение

```sql
SELECT
    id,
    scores,
    ROW_NUMBER() OVER (ORDER BY scores DESC) AS position
FROM examination;
````

## Задание 2: FULL JOIN

### Задача

Есть две таблицы:

* первая — 30 строк
* вторая — 20 строк

Выполняется `FULL JOIN`.

Нужно определить **минимальное и максимальное возможное количество строк** в результате.

### Ответ

**Минимально: 30 строк**
**Максимально: 600 строк**

### Объяснение

#### Минимальное количество строк (30)

Минимум достигается, когда:

* все строки второй таблицы находят соответствие в первой
* и при этом соответствие **1 к 1**

Тогда:

* 20 строк совпадают
* 10 строк из первой таблицы остаются без пары

Итого:

```
20 + 10 = 30 строк
```

---

#### Максимальное количество строк (600)

Максимум достигается, когда:

* все строки имеют **одинаковый ключ**
* происходит декартово перемножение

Тогда:

```
30 × 20 = 600 строк
```

## Задание 3: Покупки

### Постановка задачи

Даны таблицы:

#### `account`

* `id` — ID счёта
* `client_id` — ID клиента
* `open_dt` — дата открытия
* `close_dt` — дата закрытия

#### `transaction`

* `id` — ID транзакции
* `account_id` — ID счёта
* `transaction_date` — дата транзакции
* `amount` — сумма
* `type` — тип транзакции

---

### Требуется

Вывести `ID` клиентов, которые:

* за **последний месяц**
* по **всем своим счетам**
* совершили покупки (`type = 'PUR'`)
* на сумму **меньше 5000 рублей**

Ограничения:

* без подзапросов
* без оконных функций

---

### Решение

```sql
SELECT
    a.client_id
FROM account a
JOIN transaction t
    ON a.id = t.account_id
WHERE
    t.type = 'PUR'
    AND t.transaction_date >= CURRENT_DATE - INTERVAL '1 month'
GROUP BY
    a.client_id
HAVING
    SUM(t.amount) < 5000;
```
