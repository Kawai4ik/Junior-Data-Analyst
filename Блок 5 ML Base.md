## Задание 1: Пони тоже кони

### Условие

Есть две модели для бинарной классификации (лошади vs пони):

- Модель 1: ROC-AUC = 0.7  
- Модель 2: ROC-AUC = 0.1  

Нужно выбрать модель для дальнейшей работы и объяснить, что с ней делать.

## Ответ

Возьму вторую модель (ROC-AUC = 0.1) и инвертирую её предсказания.

## Обоснование

- ROC-AUC = 0.5 — случайная модель  
- ROC-AUC < 0.5 означает, что модель систематически ошибается  
- ROC-AUC = 0.1 

Если поменять предсказания:
- 0 → 1  
- 1 → 0  

или инвертировать вероятности:
- `p → 1 - p`

то метрика станет: 1 - 0.1 = 0.9

## Вывод

Вторая модель фактически содержит сильный сигнал, но с перевёрнутыми классами.  
После инверсии она даст значительно лучшее качество (ROC-AUC ≈ 0.9), чем первая модель (0.7).

## Задание 2: Ручной расчёт ROC-AUC

### Ответ

**ROC-AUC = 0.75**

## Ход решения (кратко)

1. Определяем:
   - положительных объектов (y = 1): **7**
   - отрицательных объектов (y = 0): **8**

2. ROC-AUC — это вероятность того, что случайный положительный объект имеет **оценку выше**, чем случайный отрицательный.

3. Считаем количество “правильных пар”:
   - для каждого положительного объекта считаем, сколько отрицательных имеют меньший score

4. Подсчёт:

| Положительный score | Побед над отрицательными |
|--------------------|-------------------------|
| 0.95               | 8                       |
| 0.85               | 7                       |
| 0.75               | 6                       |
| 0.70               | 6                       |
| 0.65               | 6                       |
| 0.60               | 6                       |
| 0.40               | 3                       |

Сумма:8 + 7 + 6 + 6 + 6 + 6 + 3 = 42


5. Общее количество пар: ROC-AUC = 42 / 56 = 0.75

## Вывод

Модель корректно ранжирует объекты в 75% случаев, что говорит о среднем качестве классификации.

## Задание 3: Ручной расчёт корреляции

### Ответ

**Коэффициент корреляции Пирсона: -0.85**

## Ход решения

1. Найдены средние значения:
   - среднее по X (чашки кофе): **2.3**
   - среднее по Y (баллы): **73.2**

2. Рассчитаны:
   - отклонения от средних  
   - сумма произведений отклонений  
   - сумма квадратов отклонений  

3. Использована формула корреляции Пирсона:r ≈ -93.6 / √(8.1 * 1499.6) ≈ -0.85

## Интерпретация результата

- Значение **-0.85** говорит о **сильной отрицательной линейной связи**:
  - чем больше кофе — тем ниже балл (в среднем)

## Причинно-следственная связь

Утверждать причинность **нельзя**, потому что:

- корреляция ≠ причинность  
- возможны скрытые факторы:
  - стресс  
  - уровень подготовки  
  - недосып  
- кофе может быть следствием (например, плохо подготовился → пьёт больше кофе)

## Вывод

- Есть сильная отрицательная корреляция  
- Делать вывод о причинно-следственной связи нельзя без дополнительного анализа
